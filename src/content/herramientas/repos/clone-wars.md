## Qué es

Clone Wars es una lista de más de 100 clones open source de las apps que usas cada día.

Airbnb. Netflix. Spotify. Instagram. TikTok. Twitter. Notion. Trello.
Cada uno construido por developers de la comunidad con código abierto.

34.600 estrellas. AGPL-3.0.

## Por qué importa

La mejor forma de aprender a desarrollar es construir cosas reales.

Pero empezar desde cero sin referencia es difícil.
Los tutoriales de "todo app" no te enseñan cómo se estructura un producto real.

Estos clones sí. Código real, estructura real, decisiones de arquitectura reales.
Y puedes modificarlos, extenderlos y aprender haciendo.

Sin pagar 500€ en un curso que te enseña un todo app de colores.

## Cómo usarlo

Es una lista en GitHub. Abres el README y buscas el clone que te interesa.

Cada entrada tiene enlace al repo del clone con instrucciones de instalación propias.

## Algunos highlights de la lista

- **Airbnb clone** - búsqueda, mapas, reservas, pagos
- **Netflix clone** - streaming, autenticación, catálogo
- **Spotify clone** - reproductor, playlists, búsqueda
- **Instagram clone** - feed, stories, mensajes directos
- **Twitter/X clone** - timeline, replies, retweets
- **Notion clone** - editor de bloques, bases de datos
- **Trello clone** - tableros Kanban, drag & drop

## Casos de uso

- Aprender un framework (Next.js, React, Vue) construyendo algo conocido.
- Entender cómo se estructura una app real antes de construir la tuya.
- Preparar entrevistas técnicas con proyectos reales en el portfolio.
- Base de código para una idea propia que comparte funciones con alguno de estos.

## Limitaciones honestas

Es una lista curada, no una plataforma. La calidad de cada clone depende de quien lo hizo.

Algunos clones están desactualizados o usan versiones viejas de frameworks.

No todos están mantenidos activamente. Revisa la fecha del último commit antes de usarlo como base.
