## Qué es

OpenCut es el editor de video que la comunidad open source construyó cuando CapCut empezó a levantar sospechas sobre qué hace con tus datos.

Editor de timeline completo. Cortes, transiciones, texto, audio.
En el navegador o instalado en local.
Sin cuenta. Sin suscripción. Sin que nadie vea tu contenido.

45.800 estrellas en GitHub. MIT.

## Por qué importa

CapCut es de ByteDance, la misma empresa detrás de TikTok.
Todo lo que editas pasa por sus servidores.

Para contenido personal o de negocio donde la privacidad importa, eso es un problema.

OpenCut hace lo mismo sin mandarte a ningún servidor chino.
Y sin cobrarte 13€ al mes cuando decidan que el tier gratuito se acabó.

## Cómo instalarlo

Versión web: abres opencut.app y editas directamente en el navegador. Sin instalación.

Para instalarlo en local:

```bash
git clone https://github.com/OpenCut-app/OpenCut.git
cd OpenCut
npm install
npm run dev
```

Se abre en `http://localhost:3000`.

## Casos de uso

- Reels y shorts para redes sociales sin mandar el contenido a CapCut.
- Ediciones rápidas de clips para YouTube sin pagar Adobe.
- Contenido de empresa donde no puedes usar herramientas que envían datos a terceros.
- Cualquiera que quiera un editor decente sin suscripción mensual.

## Qué sustituye

| Herramienta | Precio | Diferencia |
|---|---|---|
| CapCut Pro | 13€/mes | OpenCut es gratis y no rastrea |
| Adobe Premiere Rush | 10€/mes | Sin suscripción |
| Canva Video | 13€/mes | Sin límite de exportaciones |

## Limitaciones honestas

No es Premiere Pro. Para producción profesional con multicámara, color grading avanzado o efectos complejos, todavía no llega.

El proyecto es joven y algunas funciones están en desarrollo activo. Puede tener bugs.

La versión web depende del navegador para procesar el video, lo que en equipos lentos puede ir justo.
